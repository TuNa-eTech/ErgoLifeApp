# Design System Reference

## Colors
| Name | Hex | Usage |
|---|---|---|
| Soft Porcelain | `#F2F4F7` | Background |
| Pure White | `#FFFFFF` | Surface |
| Vibrant Orange | `#FF6B00` | Primary (Buttons, CTA) |
| Navy Blue | `#1D2939` | Text Primary |
| Slate Grey | `#667085` | Text Secondary |
| Eucalyptus | `#00C48C` | Success |

## Typography
- **Font**: Inter (Google Fonts) or SF Pro Rounded.
- **Style**: Bold, rounded, easy to read.
